# MEXT AI/ML Journey Planner 🗺️

> A dynamic roadmap planner for prospective graduate students pursuing an AI/ML Master's degree in Japan via the MEXT Scholarship.

## 🌟 Overview

The **MEXT Journey Planner** is a specialized platform tailored for students with a focus on Artificial Intelligence and Machine Learning. Securing the MEXT scholarship is highly competitive; this application breaks down the complex, multi-year preparation process into an actionable, trackable roadmap.

## ✨ Key Features
- **AI/ML Focus**: Specialized guidance on how to build a research proposal and profile suited for AI and Data Science laboratories in Japan.
- **Interactive Timeline**: Visual roadmap tracking key milestones (JLPT exams, contacting professors, embassy submissions).
- **Task Management**: Built-in checklists ensuring no critical deadline is missed during the application cycle.
- **Responsive Design**: Accessible on any device to keep your planning mobile.

## 🛠️ Technology Stack
- **Framework**: React / Vite
- **Language**: TypeScript
- **UI Architecture**: Component-based architecture with Tailwind utility classes.

## 🚀 Getting Started

To explore the journey planner locally:

1. **Clone the repo:**
   ```bash
   git clone https://github.com/MRYASHYT/youknowtrack.git
   cd youknowtrack
   ```

2. **Install the required packages:**
   ```bash
   npm install
   ```

3. **Start the local server:**
   ```bash
   npm run dev
   ```
